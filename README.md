# William Greenwalt Portfolio

A modern, responsive portfolio website for William Greenwalt - UX and Product Experience Designer with nearly 20 years of experience in tech.

## Features

- **Modern Design**: Clean, professional design with a dark theme and green accent colors
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Subtle animations and transitions for enhanced user experience
- **Interactive Navigation**: Smooth scrolling navigation with active section highlighting
- **Project Showcase**: Beautiful grid layout for featured projects
- **Skills Section**: Comprehensive display of technical and design skills
- **Contact Section**: Easy ways to get in touch

## Technologies Used

- **HTML5**: Semantic markup for accessibility and SEO
- **CSS3**: Modern CSS with Grid, Flexbox, and custom properties
- **JavaScript**: Vanilla JavaScript for interactions and animations
- **Google Fonts**: Inter font family for optimal typography

## Getting Started

### Prerequisites
- Node.js and npm (recommended) OR
- Python 3 (alternative) OR
- Any local web server

### Option 1: Using Node.js (Recommended)
1. Install Node.js from [nodejs.org](https://nodejs.org/)
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:3000`

### Option 2: Using Python
1. Open terminal in the project directory
2. Run:
   ```bash
   python -m http.server 3000
   ```
3. Open your browser and navigate to `http://localhost:3000`

### Option 3: Using Any Web Server
Simply open `index.html` in your preferred web server or IDE's live preview feature.
